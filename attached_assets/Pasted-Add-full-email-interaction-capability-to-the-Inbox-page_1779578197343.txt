## ZOHO MAIL API SCOPE REQUIREMENTS

Check that these OAuth scopes are requested during 
connection:
- ZohoMail.messages.ALL (read + send + delete)
- ZohoMail.folders.ALL (archive, move)
- ZohoMail.accounts.READ (account info)

If current scope is missing send/delete/folder access,
update the OAuth authorization URL to include full scopes
and show a "Reconnect to enable full email actions" banner
if current token has insufficient scope.

## ZOHO MAIL API CALLS TO IMPLEMENT

Read full message:
GET /api/accounts/{accountId}/messages/{messageId}

List thread messages:
GET /api/accounts/{accountId}/threads/{threadId}/messages

Send email:
POST /api/accounts/{accountId}/messages
Body: { toAddress, subject, content, mailFormat: "html" }

Mark read/unread:
PUT /api/accounts/{accountId}/updatemessage
Body: { messageId, isRead: true/false }

Move to folder (archive/trash):
PUT /api/accounts/{accountId}/updatemessage  
Body: { messageId, folderId: {archive_folder_id} }
