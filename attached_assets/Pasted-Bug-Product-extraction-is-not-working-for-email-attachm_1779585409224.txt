When Extract Products is triggered for an RFQ, fetch
all attachments from Zoho Mail before calling Claude.

Zoho Mail API to get attachment content:

Step 1 — Get message details including attachment metadata:
GET https://mail.zoho.com/api/accounts/{accountId}/messages/{messageId}
Header: Authorization: Zoho-oauthtoken {access_token}

Response includes attachments array:
[{
  "attachmentId": "xxx",
  "attachmentName": "RFQ_Products.pdf",
  "attachmentSize": 45000,
  "attachmentType": "application/pdf"
}]

Step 2 — Download attachment content:
GET https://mail.zoho.com/api/accounts/{accountId}/messages/{messageId}/attachments/{attachmentId}
Header: Authorization: Zoho-oauthtoken {access_token}

Returns raw file bytes.

Implement a function:
fetchEmailAttachments(accountId, messageId, accessToken)
